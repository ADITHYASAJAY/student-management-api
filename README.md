# Student Management API

A RESTful Web API built using **ASP.NET Core** and **Entity Framework Core**, following clean architecture principles.

## 🚀 Tech Stack
- C# / .NET 7
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server (LocalDB)
- Swagger (OpenAPI)

## 🧱 Architecture
- Controllers
- Services (Business Logic)
- Repositories (Data Access)
- DTOs (Data Transfer Objects)

## ✨ Features
- CRUD operations for Student entity
- Async API endpoints
- Clean separation of concerns
- Swagger UI for API testing

## 📌 Endpoints
- GET `/api/students`
- POST `/api/students`

## ▶️ Run Locally
1. Clone the repository
2. Update connection string in `appsettings.json`
3. Run EF migrations
4. Press **F5** to run
5. Open `/swagger`
